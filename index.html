Download phrased!

Phrased is a small utility to create secure and pleasant passphrases from a selection of predefined dictionaries. Besides this online generator, Phrased is available for many platforms for direct installation:

Installation: Mac OSX

The latest release is automatically pushed as a homebrew formular which can be tapped and installed like this: 

brew tap phrased-org/tap
brew install phrased

Manual Installation

Phrased is also available as a downloadable binary from GitHub for Linux, Windows and Mac OSX.
